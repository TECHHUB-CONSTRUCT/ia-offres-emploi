# 🤖 Agent IA de Recherche d'Emploi

## Description
Agent automatique qui :
- Analyse votre CV
- Parcourt internet (Indeed, Emploi.cm)
- Trouve les offres correspondant à votre profil
- Vous envoie un email quotidien
- Génère une interface web

## Installation sur votre PC

1. **Installer PHP** (téléchargez depuis windows.php.net)
2. **Cloner le dépôt** :
```bash
git clone https://github.com/VOTRE_USERNAME/ia-offres-emploi-php.git
cd ia-offres-emploi-php
